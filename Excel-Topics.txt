6. Mention the order of operations used in Excel while evaluating formulas.
	Excel follows a fixed rule called 'Operator Precedence'.
	The Order is
		1.Parentheses "()": Operations inside brackets are done first.
		2.Exponents "^": 2^3=8
		3.Multiplication(*) & Division(/):Evaluate left to right.
		4.Addition(+) & Substraction(-): Evaluate left to right.  

8. What is the difference between count, counta, and countblank?
	COUNT:Counts only numeric values(Numbers,Dates,Time Values).
		Formula: =COUNT(range) e.g.: =COUNT(A1:A4)
	COUNTA: Counts all non-empty cells(Num, Text, Dates, 	Formulas(even if results is empty text "")).
		Formula: =COUNTA(range)
	COUNTBLANK: Counts only empty cells. It could be truly blank cells, 			Cell with formulas returning empty text("")
		Formula: =COUNTBLANK(range)

14. How is VLOOKUP different from the LOOKUP function?

LOOKUP is a basic and old function.

	Searches for a value in one row or one column

	Data must be sorted

	Limited control

	Rarely used in real work

VLOOKUP is a more advanced and widely used function.

	Searches vertically (top to bottom)

	Works with unsorted data (using FALSE)

	Can return data from any column to the right

	Used heavily in offices

LOOKUP requires sorted data and has limited use, while VLOOKUP is more flexible and widely used for vertical searches.

26. What are the wildcards available in Excel?
	Excel has three wildcards: asterisk (*), question mark (?), and tilde (~).

1Ô∏è‚É£ Asterisk *

Represents any number of characters (including zero)

Example:

A* ‚Üí matches Apple, Account, Alex

*ing ‚Üí matches Running, Coding

2Ô∏è‚É£ Question Mark ?

Represents exactly one character

Example:

T?t ‚Üí matches Test, Text

A?C ‚Üí matches ABC, ADC

3Ô∏è‚É£ Tilde ~

Used to escape wildcards (treat them as normal characters)

Example:

~* ‚Üí finds an actual *

~? ‚Üí finds an actual ?
