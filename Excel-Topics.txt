2. What do you mean by Relative cell referencing and Absolute cell referencing in MS Excel?
	Relative cell reference is a cell address that changes automatically, When we copy a formula to another cell. if you copy a fromul from A3 to B3 and the formula is =A1+A2. So in B3 automatically this formula will be B1+B2
#Excel adjust the reference relative to the new position.
	Absolute cell reference means a cell reference that doesn NOT change.When you copy or drag aformula to another cell. To achieve this we use $ sign. E.g:$A$1

6. Mention the order of operations used in Excel while evaluating formulas.
	Excel follows a fixed rule called 'Operator Precedence'.
	The Order is
		1.Parentheses "()": Operations inside brackets are done first.
		2.Exponents "^": 2^3=8
		3.Multiplication(*) & Division(/):Evaluate left to right.
		4.Addition(+) & Substraction(-): Evaluate left to right.  

8. What is the difference between count, counta, and countblank?
	COUNT:Counts only numeric values(Numbers,Dates,Time Values).
		Formula: =COUNT(range) e.g.: =COUNT(A1:A4)
	COUNTA: Counts all non-empty cells(Num, Text, Dates, 	Formulas(even if results is empty text "")).
		Formula: =COUNTA(range)
	COUNTBLANK: Counts only empty cells. It could be truly blank cells, 			Cell with formulas returning empty text("")
		Formula: =COUNTBLANK(range)
