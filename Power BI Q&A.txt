* A Data Model is the structure of tables and relationships used to analyze data in Power BI.

* Star Schema is a data modelling technique where a central fact table is connected to multiple dimension tables.

* A Dashboard is a single page view of key metrics created in Power BI Service.

* A Slicer is a visual filter used to filter data interactively.

* CALCULATE() function modifies filter context and perform calculations in DAX.

* RLS(Row Level Security) restricts data access for users based on roles.

* A Gateway connects on-premise data sources to power BI service.

* Incremental refresh updates only new or changed data instead of refreshing the entire dataset.

* KPIs(Key Performance Indicators) measure business performance like sales, profit, growth.
